\chapter{Bounding Boxes} \label{Bbox}


Bounding boxes are used to approximate objects and the only operations 
you can perform on them is to get their bounds and to check if two
bounding boxes overlap. They are axis-parallel and the difference to 
the class \CCstyle{CGAL_Iso_rectangle_2<R>} is that they are not templated.

Here comes a typical situation where to use them: In order to check
if two complicated polygons intersect you first check if their 
respective bounding boxes intersect. If the answer is negative,
you are done, otherwise you really have to check if the polygons 
intersect.
 
\begin{class} {CGAL_Bbox_2}
\CCsection{2D Bbox}

\definition  An object \CCstyle{b} of the class \classname\ is a bounding 
box in the two-dimensional Euclidean plane $\E_2$. This class is not templated.


\creation
\creationvariable{b}

\CCstyle{#include <CGAL/Bbox_2.h>}

\constructor{CGAL_Bbox_2(double x_min, double y_min,
                         double x_max, double y_max);}
            {introduces a bounding box \var\ with lower left corner at 
             \CCstyle{(xmin, ymin)} and with upper right corner at 
             \CCstyle{(xmax, ymax)}.}


\operations
\threecolumns{5cm}{4cm}

\method{bool operator==(const CGAL_Bbox_2 &c) const;}
       {Test for equality.}

\method{bool operator!=(const CGAL_Bbox_2 &q) const;}
       {Test for inequality.}

\method{double                 xmin() const;}
       {}
\method{double                 ymin() const;}
       {}
\method{double                 xmax() const;}
       {}
\method{double                 ymax() const;}
       {}

\method{CGAL_Bbox_2              operator+(const CGAL_Bbox_2 &c) const;}
       {returns the bounding box of \var\ and \CCstyle{c}.}

\function{bool CGAL_do_overlap(const CGAL_Bbox_2 &bb1, const CGAL_Bbox_2 &bb2);}
       {}
\end{class}
