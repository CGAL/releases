The 3D Kernel: an Overview

This chapter presents an overview of the three-dimensional kernel of CGAL. The kernel consists of elementary 3D objects (such as points, lines, planes, etc.) and of predicates on these objects (such as coordinate comparison, orientation, etc.). We also provide affine transformations on these objects. These classes are all templated by the representation class R, which is currently either CGAL_Cartesian<FT> or CGAL_Homogeneous<RT> for a so-called field tupe FT, and ring type RT, see Chapter .
