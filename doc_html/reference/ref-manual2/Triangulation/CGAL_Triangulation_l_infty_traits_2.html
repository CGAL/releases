L\infty Metric

Definition

The class CGAL_Triangulation_l_infty_traits_2<R> is used for the L\infty metric for 2D points. A ``circle'' for that metric is a square with sides parallel to the axis, and the Delaunay triangulation is, as usual, a triangulation such that all the ``circles'' circumscribing the triangles are empty.

Degeneracies are solved by a symbolic perturbation scheme. Point (x,y) is perturbed in (x+eps y,y+ eps2 x) for eps \rightarrow 0+. This perturbation, solves all degeneracies, such that points horizontally or vertically aligned...
Precondition: In the current version, the L\infty Delaunay triangulation do not work for any set of points. The set of points must satisfied the requirement that the hull of the Delaunay triangulation is equal to the convex hull of the points. An easy way to satisfy this precondition is to first insert four points as a bounding box of the set of points. Due to the perturbation technique, the four points cannot be inserted in any order, for an isothetic square, the upper right corner should be inserted last.
