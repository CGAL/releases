Traits (Traits)

Definition

A class Traits that satisfies the requirements of a triangulation traits class must provide the following types and operations.

Types

Traits::Vertex
A type to hold a vertex in a triangulation. The requirements for a triangulation vertex from Section must be fulfilled.

Traits::Face
A type to hold a face in a triangulation. The requirements for a triangulation face from Section must be fulfilled.

Traits::Point
The type must provide a copy constructor, assignment and equality test.

Traits::Segment
The type must provide a constructor that takes two points as argument.

Traits::Triangle
The type must provide a constructor that takes three points as argument.

Traits::Distance
Only CGAL_Delaunay_triangulation_2<Traits>::nearest_vertex(..) needs this type. The requirements for a Distance type must be fulfilled, see Section below.

Creation

Only a default constructor is required. Note that further constructors can be provided.

Traits t;
A default constructor.

Operations

CGAL_Comparison_result t.compare_x ( Point p0, Point p1)
Compares the x-coordinates.
CGAL_Comparison_result t.compare_y ( Point p0, Point p1)
Compares the y-coordinates.
CGAL_Orientation t.orientation ( Point p0, Point p1, Point p2)
Performs the orientation test.
CGAL_Orientation t.extremal ( Point p0, Point p1, Point test)
Returns the constant CGAL_ON_POSITIVE_SIDE if the three points p0, p1 and test define a unique circumscribing circle, and are oriented counterclockwise along this circle.\quad Returns the constant CGAL_ON_ORIENTED_BOUNDARY or CGAL_ON_NEGATIVE_SIDE, otherwise.
CGAL_Oriented_side
t.side_of_oriented_circle ( Point p0,
Point p1,
Point p2,
Point test)
Performs the incircle test. Points p0, p1 and p2 must be oriented counterclockwise and the method tests point test with respect to the circumscribing circle.
