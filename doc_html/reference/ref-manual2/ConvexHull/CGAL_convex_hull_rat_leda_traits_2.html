A Convex Hull Traits Class for two-dimensional LEDA ( CGAL_convex_hull_rat_leda_traits_2)

Definition

The class CGAL_convex_hull_rat_leda_traits_2 is a convex hull traits class for the two-dimensional rational geometry provided in LEDA, especially rat_points. It provides only the types listed below, especially, no Less_dist_to_line trait is provided. Therefore, this traits class cannot be used with CGAL_ch_eddy(). The reason for the incompleteness is simply, that LEDA does not provide appropriate predicates for this special case.

#include < CGAL/convex_hull_rat_leda_traits_2.h>
