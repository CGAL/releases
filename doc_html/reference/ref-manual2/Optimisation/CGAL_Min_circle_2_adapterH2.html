Traits Class Adapter for 2D Smallest Enclosing Circle to 2D Homogeneous Points (CGAL_Min_circle_2_adapterH2)

Definition

The class CGAL_Min_circle_2_adapterH2<PT,DA> interfaces the 2D optimisation algorithm for smallest enclosing circles with the point class PT. The data accessor DA [K{ü}hl 97] is used to access the hx-, hy- and hw-coordinate of PT, i.e. PT is supposed to have a homogeneous representation of its coordinates.

#include < CGAL/Min_circle_2_adapterH2.h>

Example

The following example illustrates the use of the traits class adapter with your own point class. For the sake of simplicity, we expect the point class to have int coordinates with access functions hx(), hy() and hw(). Based on this we show how to implement and use a data accessor for the point class.

The requirements catalogs of the point class and the data accessor are listed in Section .

Note: In this release correct results are only guaranteed if exact arithmetic is used, so this example (using integer arithmetic with possible overflows) is only intended to illustrate the techniques. 

  #include <CGAL/Min_circle_2_adapterH2.h>
  #include <CGAL/Min_circle_2.h>

  /* your own point class */
  class Pt {
      ...
    public:
      ...
      Pt( int hx, int hy, int hw);
      ...
      int  hx( ) const;
      int  hy( ) const;
      int  hw( ) const;
      ...
  };

  /* the data accessor for Pt */
  class Pt_DA {
    public:
      typedef  int  Rt;
      void  get( Pt const& p, int& hx, int& hy, int& hw) const
                { hx = p.hx(); hy = p.hy(); hw = p.hw(); }
      int  get_hx( Pt const& p) const { return( p.hx()); }
      int  get_hy( Pt const& p) const { return( p.hy()); }
      int  get_hw( Pt const& p) const { return( p.hw()); }
      void  set( Pt& p, int hx, int hy, int hw) const { p = Pt( hx, hy, hw); }
  };

  /* some typedefs */
  typedef  CGAL_Min_circle_2_adapterH2< Pt, Pt_DA >  Adapter;
  typedef  CGAL_Min_circle_2< Adapter >              Min_circle;

  /* do something with Min_circle */
  Min_circle  mc(...);
