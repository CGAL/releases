2D Parabola (CGAL_Parabola_2)

This class exist only for Cartesian representations.

Definition

An object par of the class CGAL_Parabola_2 is a parabola in the two-dimensional Euclidean plane E2. It is defined by a base point p, a vector v defining an angle of rotation and a curvature c. The parabola is the set of points q, for which there is a lambda with q = p + lambda \, v + lambda2 \, c \, v\perp, where vector v\perp is orthogonal to v and rotated in counterclockwise direction.

We call the line to which the points of the parabola have the same distance as to the focus the director of the parabola. The director is directed such that the focus is on the same side of the parabola and the line.

Creation

#include <CGAL/Parabola_2.h>

CGAL_Parabola_2<R> par ( CGAL_PointC2<FT> p,
CGAL_Vector_2<R> v,
R::RT c);
introduces a parabola par with the base at p. It is rotated around the base of the parabola by the angle between the positive x-axis and the direction of vector v. It has a curvature c.

CGAL_Parabola_2<R> par ( CGAL_Line_2<R> l, CGAL_Point_2<R> p);
introduces a parabola par which is the bisector between a line l and a point p.

Operations

bool par == p Test for equality: two parabolas are equal, iff the equations are the same.
bool par != p Test for inequality.
bool par.is_degenerate () Parabola par is degenerate, iff the curvature of par is 0.
CGAL_Point_2<R> par.base () returns the base point of the parabola.
CGAL_Point_2<R> par.focus () returns the focus point of the parabola.
CGAL_Line_2<R> par.director () returns the director of par.
CGAL_Vector_2<R> par.vector () returns the vector v of the parametrized equation of par.
R::RT par.curvature () returns the curvature of par.
CGAL_Parabola_2<R> par.opposite () returns the parabola oriented in the opposite direction.
CGAL_Point_2<R> par ( R::FT lambda) returns the point at par.base() + lambda par.vector() + lambda2 \, par.curvature( ) \, par.vector() \perp.
int
par.lambdas_at_x ( R::FT x,
R::FT &lambda1,
R::FT &lambda2)
computes parameters lambda1 and lambda2, for which par(lambda1).x()==x, par(lambda2).x()==x. The return value is 0, 1 or 2 and it gives the number of solutions. The value of lambdai is undefined if the return value is smaller than i.
int
par.lambdas_at_y ( R::FT y,
R::FT &lambda1,
R::FT &lambda2)
computes parameters lambda1 and lambda2, for which par(lambda1).y()==y, par(lambda2).y()==y. The return value is 0, 1 or 2 and it gives the number of solutions. The value of lambdai is undefined if the return value is smaller than i.
R::FT par.x_at_lambda ( R::FT lambda)
returns par(lambda).x().
R::FT par.y_at_lambda ( R::FT lambda)
returns par(lambda).y().
CGAL_Point_2<R> par.projection ( CGAL_Point_2<R> p)
returns the projection of p on the parabola in the direction perpendicular to the director.
void
par.projection ( CGAL_Point_2<R> p,
R::FT &lambda)
as the function above. Additionally it assigns the value lambda to the reference parameter lambda, such that par.(lambda) == par. projection(p).
CGAL_Oriented_side par.oriented_side ( CGAL_Point_2<R> p)
returns CGAL_ON_POSITIVE_SIDE, CGAL_ON_ORIENTED_BOUNDARY or CGAL_ON_NEGATIVE_SIDE, depending where point p is relative to the oriented parabola par.
bool par.has_on_positive_side ( CGAL_Point_2<R> p)
bool par.has_on_negative_side ( CGAL_Point_2<R> p)
bool par.has_on_boundary ( CGAL_Point_2<R> p)
CGAL_Parabola_2<R>
par.transform ( CGAL_Aff_transformation_2<R> t)
returns the parabola obtained by applying t on the director and on the focus of par.
