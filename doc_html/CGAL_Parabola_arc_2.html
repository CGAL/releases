2D Parabola Arc (CGAL_Parabola_arc_2)

This class exist only for Cartesian representations.

Definition

An object arc of the class CGAL_Parabola_arc_2 is a parabola arc in the two-dimensional Euclidean plane E2. It is defined by its supporting parabola and two parameters lambda1 and lambda2 of the supporting parabola par and it is oriented from par (lambda1) to par (lambda2).

Creation

#include <CGAL/Parabola_arc_2.h>

CGAL_Parabola_arc_2<R> arc ( CGAL_Parabola_2<R> par,
R::RT lambda_1,
R::RT lambda_2);
introduces a parabola arc arc of the parabola par between points par(lambda1) and par(lambda2).

Operations

bool arc == a Test for equality: two parabola arcs are equal, iff the supporting parabolas and the sources and targets are equal.
bool arc != a Test for inequality.
bool arc.is_degenerate () A parabola arc arc is degenerate, if the supporting parabola is degenerate, or if source and target are equal.
CGAL_Parabola_2<R> arc.supporting_parabola ()
returns the supporting parabola of arc.
CGAL_Point_2<R> arc.source () returns the source of arc.
CGAL_Point_2<R> arc.target () returns the target of arc.
CGAL_Point_2<R> arc.vertex ( int i) vertex(0) returns the source, vertex(1) returns the target of arc. The parameter i is taken modulo 2, which gives easy access to the other end point.
CGAL_Point_2<R> arc [ int i] returns vertex(i).
R::RT arc.lambda_source () returns the lambda of the source of arc.
R::RT arc.lambda_target () returns the lambda of the target of arc.
R::RT arc.lambda ( int i) returns the lambda of source or target of arc: lambda(0) returns the lambda of the source, lambda( 1) returns the lambda of the target of arc. The parameter i is taken modulo 2, which gives easy access to the lambda of the other end point.
bool arc.has_on ( CGAL_Point_2<R> p)
A point is on arc, iff it is equal to the source or target of arc, or if it is in the interior of arc.
CGAL_Parabola_arc_2<R> arc.opposite () returns a parabola arc with source and target interchanged.
CGAL_Bbox_2 arc.bbox () returns the smallest bounding box containing arc.
