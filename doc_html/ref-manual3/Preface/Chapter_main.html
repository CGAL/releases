Preface

CGAL is a Computational Geometry Algorithms Library written in C++, which is developed by the Esprit project CGAL. This project is carried out by a consortium consisting of Utrecht University (The Netherlands), ETH Zürich (Switzerland), Freie Universiät Berlin (Germany), INRIA Sophia-Antipolis (France), Max-Planck Institut für Informatik, Saarbrücken (Germany), RISC Linz (Austria) and Tel-Aviv University (Israel). You find more information on the project on the CGAL home page .

Should you have any questions, comments, remarks or criticism concerning CGAL, please send a message to <cgal@cs.uu.nl>.
