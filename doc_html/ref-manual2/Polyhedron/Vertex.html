Vertex of a Polyhedral Surface

Definition

A vertex optionally stores a point and a reference to an incident halfedge that points to the vertex. Type tags as defined in CGAL_Polyhedron_3 indicate whether these member functions are supported. Figure  depicts the relationship between a halfedge and its incident halfedges, vertices, and facets.

Types

Vertex defines the same types as CGAL_Polyhedron_3 except the traits class.

Access Functions

Halfedge_handle v.halfedge () an incident halfedge pointing to v.
Point& v.point () the point.
Halfedge_around_vertex_circulator
v.vertex_begin () circulator of halfedges around the vertex (clockwise).
