Models of Polyhedron_traits

Definition

CGAL_Polyhedron_default_traits_3<R> is a model of the Polyhedron_traits concept from Section  based on the CGAL-kernel. It defines CGAL_Point_3<R> as points, CGAL_Vector_3<R> as surface normal vectors, and CGAL_Plane_3<R> as plane equations.

#include <CGAL/Polyhedron_default_traits_3.h>
