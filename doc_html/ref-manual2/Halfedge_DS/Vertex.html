Requirements for a Vertex

Definition

A vertex optionally stores a point and a reference to an incident halfedge that points to the vertex. Type tags indicate whether these member functions are supported. Figure  depicts the relationship between a halfedge and its incident halfedges, vertices, and facets.

Type for (optionally) associated geometry. If a point is not supported the type is void*.
