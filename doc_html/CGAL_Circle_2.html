2D Circle (CGAL_Circle_2)

Definition

An object c of the class CGAL_Circle_2 is a circle in the two-dimensional Euclidean plane E2. A circle is the set of points in E2 that have distance sqrt( c.squared_radius() ) from c.center() of the circle.

An object c of the class CGAL_Circle_2 is oriented, i.e., it has clockwise or counterclockwise orientation. circle (resp. its boundary) splits E2 in a positive and a negative side, where the positive side is to the left of the boundary. Furthermore it splits E2 in a bounded and an unbounded side.

Creation

#include <CGAL/Circle_2.h>

CGAL_Circle_2<R> c ( CGAL_Point_2<R> p,
CGAL_Point_2<R> q,
CGAL_Point_2<R> r);
introduces a circle c which passes through the points p, q and r. The orientation of c is the orientation of p, q, and r. If the three points p, q, and r are collinear, the circle is degenerate where the center is undefined and the squared radius is zero.

CGAL_Circle_2<R> c ( CGAL_Point_2<R> center,
R::FT squared_radius);
introduces a circle c, defined by a center, a squared_radius and positive orientation, i.e. positive side and bounded side coincide.

CGAL_Circle_2<R> c ( CGAL_Point_2<R> center,
R::FT squared_radius,
CGAL_Orientation orientation);
introduces a circle c, with midpoint center, a squared_radius and an orientation.

CGAL_Circle_2<R> c ( CGAL_Point_2<R> center,
CGAL_Point_2<R> p);
introduces a circle c with midpoint center and positive orientation, such that p lies on c.

CGAL_Circle_2<R> c ( CGAL_Point_2<R> center,
CGAL_Point_2<R> p,
CGAL_Orientation orientation);
introduces a circle c with midpoint center and orientation orientation, such that p lies on c.

Operations

bool c == c2 Test for equality: two circles are equal, iff they have the same center, radius and orientation.
bool c != c2 Test for inequality.
CGAL_Point_2<R> c.center () returns the center of c.
R::RT c.squared_radius () returns the squared radius of c.
CGAL_Orientation c.orientation () returns the orientation of c.
bool c.is_degenerate () circle c is degenerate, iff the squared radius equals zero.
CGAL_Oriented_side c.oriented_side ( CGAL_Point_2<R> p)
returns CGAL_ON_POSITIVE_SIDE, CGAL_ON_ORIENTED_BOUNDARY or CGAL_ON_NEGATIVE_SIDE, depending on where point p is.
bool c.has_on_boundary ( CGAL_Point_2<R> p)
bool c.has_on_positive_side ( CGAL_Point_2<R> p)
returns true, if p is to the left of the oriented boundary of c.
bool c.has_on_negative_side ( CGAL_Point_2<R> p)
returns true, if p is to the right of the oriented boundary of c.
CGAL_Bounded_side c.bounded_side ( CGAL_Point_2<R> p)
returns true, if p is inside of c.
bool c.has_on_bounded_side ( CGAL_Point_2<R> p)
returns true, if p is on the boundary of c.
bool c.has_on_unbounded_side ( CGAL_Point_2<R> p)
returns true, if p is outside of c.
CGAL_Circle_2<R> c.opposite () returns a circle where the boundary is oriented the other way round (this flips the positive and the negative side, but not the bounded and unbounded side).
CGAL_Bbox_2 c.bbox () returns a bounding box containing c.
CGAL_Circle_2<R>
c.orthogonal_transform ( CGAL_Aff_transformation_2<R> at)
returns the circle obtained by applying at on c. Precondition: at is an orthogonal transformation

Implementation

A circle is internally represented as a point, a squared radius and an orientation.
