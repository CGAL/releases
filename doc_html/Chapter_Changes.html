Changes from Rel. 0.4 to Rel. 0.5

  1. CGAL_Cartesian replaces C.

  2. CGAL_Homogeneous replaces H.

  3. #include <CGAL/Cartesian.h> replaces #include <CGAL/cartesian.h>.

  4. #include <CGAL/Homogeneous.h> replaces #include <CGAL/homogeneous.h>.

  5. CGAL_Homogeneous has only one parameter namely the ring type.

  6. The global function bool CGAL_identical(CGAL_Class &, CGAL_Class &) replaces the member functions bool CGAL_Class::identical(CGAL_Class &).

  7. In class CGAL_Segment_2: source()/target() replaces start()end().

  8. In class CGAL_Ray_2: source() replaces start().

  9. We introduced CGAL_Oriented_side and CGAL_Bounded_side which replace CGAL_Oriented_side.

  10. Most predicates that started with is_ now start with has_.

  11. Intersection computations have now a more elegant mechanism.
