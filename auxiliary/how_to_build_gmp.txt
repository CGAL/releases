This file contains information on how to build GMP library.


WINDOWS
  - get MinGW compiler and install it
  - get MSYS and install it (MSYS installation will identify the path to MINGW)
  - start MSYS shell
    - cd gmp-4.1.3
    - ./config.guess (this will give you the name of the system 
                                     e.g. athlon-pc-mingw32)
    - ./configure --build=none-pc-mingw32 --disable-static --enable-shared 
                  --prefix=/d/work/testgmp --enable-mpfr
    - make
    - make install
    - cd gmp-4.1.3/.libs
    - lib is the lib tool of Microsoft VC++
    - lib /out:gmp.lib /def:libgmp-3.dll_def
    - cd gmp-4.1.3/mpfr
    - lib /out:mpfr.lib *.o
